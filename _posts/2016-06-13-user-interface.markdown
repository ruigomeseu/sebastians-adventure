Sebastian's Adventure is a 3rd Person game, using the WASD key combination for movement.

## Keys

| Key   | Function       |
|:-----:|:--------------:|
| W     | Move Forward   |
| A     | Strafe Left    |
| S     | Move Backwards |
| D     | Strafe Right   |
| I     | Inventory      |
| E     | Action         |
| Space | Jump           |
| Shift | Sprint         |
